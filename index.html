Cyber Awareness Simulation Platform

This platform helps users identify phishing scams in a safe, controlled environment.

This is a safe, educational simulation platform. No personal data is collected and no real credentials are required.

What you will learn

  • How phishing emails look
  • How scam messages create urgency
  • How to avoid clicking malicious links

What to expect

  • You will see examples of scam messages
  • You will not be asked for real personal information
  • You will learn how to identify phishing attempts
