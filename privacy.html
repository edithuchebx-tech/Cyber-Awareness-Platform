Privacy Policy

This Privacy Policy explains how this Phishing Awareness Simulation Platform collects, uses, and protects user information.

1. Data We Collect

We may collect the following information during use of the platform:

  • Display name or alias (if provided by the user)
  • Simulation performance results (e.g. accuracy score)
  • Scenario interaction data (stored locally in the browser)

2. Purpose of Data Collection

Data is collected solely for educational purposes, to simulate phishing awareness training and track user performance across scenarios.

3. Data Storage

Data is stored locally in the user’s browser using local storage technology. No sensitive personal data is transmitted to external servers in the current version of this platform.

4. Data Sharing

We do not sell, share, or distribute user data to third parties.

5. Security

This platform is designed as a training simulation. Users are advised not to enter real passwords or sensitive personal information.

6. Your Rights

Users may clear their data at any time by using the “Clear History” feature in the results dashboard or clearing browser storage.

7. Future Enhancements

Future versions may introduce secure authentication and backend storage using industry-standard services such as Firebase or Supabase.

8. Contact

For questions regarding this simulation platform, please contact the system administrator.
